MiddleButtonTouch
=================
Quick start guide
-----------------
1.	Navigate chrome to [chrome://extensions](chrome://extensions)
2.	Click the check box to enable Developer mode.
3.	Click load unpacked extension. 
4.	Navigate to MiddleButtonTouch/Source.
5.	Click ok.
6.	Whenever you modify the source just cmd-r on the extension page and also reload the page you're testing on.
